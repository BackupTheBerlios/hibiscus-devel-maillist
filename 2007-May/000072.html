Hi Heiner,

> PS: Was ist eigentlich mit dem Umstieg auf Java 1.5?

Mhh, ich hab da hier ein Problem mit einem Kunden
fuer den ich eine Reihe Jameica-Plugins programmiert
habe. Ein Wechsel auf Java 1.5 ist dort nicht so einfach
moeglich, da die Software auf mehreren Servern laeuft
und dort auch noch andere Java-Programme installiert
sind und die dann vielleicht Probleme bereiten koennten.

Die Problematik war ja gewesen, dass JVerein Java 1.5
voraussetzt, User aber bisweilen 1.4 verwenden, da
das laut Jameica-Webseite als Mindestvoraussetzung
genuegt.

Vorschlag: Mit der kommenden Release schreibe ich
in die Dokumentation, das Java 1.5 noetig ist, verwende
selbst aber weiterhin (vorerst) 1.4 fuer die Entwicklung.
Offiziell wird dann also 1.5 verlangt - allerdings
funktioniert Jameica und Hibiscus auch noch unter 1.4
Was meinst du?
