Hallo,

ein Nachtrag zu dem Thema des Dialogs. Das Problem ist, dass der  
Dialog HINTER dem Hauptfenster erscheint. Ein erneuter Klick auf den  
Button dupliziert den Inhalt des Dialogs und zieht ihn in den  
Vordergrund. Somit wird er sichtbar.

Das Problem lässt sich lösen, indem man in der Klasse AbstractDialog,  
in der Methode open() in Zeile 419 einen Call auf shell.forceActive()  
einfügt. Das dürfte auch für die anderen Betriebssystem kein Problem  
sein, hier dann nur ein unnötiger Call...

viele Grüße
Jürgen

418	shell.open();
419	shell.forceActive();
420	while (shell != null && !shell.isDisposed()) {
421		if (!display.readAndDispatch()) display.sleep();
422	}
